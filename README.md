# GreenNest

GreenNest is a modern indoor plants e-commerce web app built with React, React Router, Tailwind CSS, and Firebase Authentication. It provides users with a seamless and accessible experience for exploring, discovering, and shopping for indoor plants that bring vibrance and serenity to their living spaces.

Visit: https://green-nest-indoor-plants.netlify.app/

## Features
- Interactive UI for browsing plant collections
- Firebase user authentication (Sign up, Sign in, Forgot Password)
- Update Profile
- Fully responsive design (mobile-first approach)
- Green-themed UI to match the store's purpose
